# Day 001 - DOM Mutation Tracker

## Overview

A JavaScript tool that monitors and logs DOM changes in real time using the MutationObserver API.

## Features

- Track added elements
- Track removed elements
- Track text modifications
- Real-time mutation log
- Clear log functionality

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- MutationObserver API

## Learning Outcomes

- DOM Manipulation
- Event Handling
- MutationObserver API
- Dynamic UI Updates
