# Day 02 - Password Strength Checker

## Overview

A real-time password strength checker that evaluates user input and provides instant feedback based on security rules.

This feature is commonly used in authentication systems to improve password security and user experience.

---

## Features

- Live password strength detection
- Strength categories: Weak, Medium, Strong
- Rule-based validation system
- Visual strength indicator bar
- Real-time checklist feedback
- Responsive UI feedback

---

## Password Rules

A strong password must include:

- At least 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one special character
