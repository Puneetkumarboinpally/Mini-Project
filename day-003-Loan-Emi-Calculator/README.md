# Day 03 - Loan EMI Calculator

## Overview

A financial calculator that helps users estimate monthly loan repayments, total interest payable, and total repayment amount based on loan details.

This type of calculator is commonly used by banks, mortgage providers, fintech applications, and personal finance platforms.

---

## Features

- Calculate Monthly EMI
- Calculate Total Interest Payable
- Calculate Total Repayment Amount
- Input Validation
- Responsive User Interface
- Financial Formula Implementation

---

## Inputs

- Loan Amount
- Annual Interest Rate
- Loan Tenure (Years)

---

## Outputs

- Monthly EMI
- Total Interest
- Total Payment

---

- CSS3
- JavaScript (ES6)
